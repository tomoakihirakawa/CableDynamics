# yuri_bridge example data

このフォルダには、橋梁ケーブル系の解析データと思われる Excel ファイルが入っている。
Excel 内には橋名・所在地・設計者などのメタ情報は見当たらないため、ここではフォルダ名に従って `yuri_bridge` と呼ぶ。

## ファイル

- `C1ケーブル座標と軸力.xlsx`
  - C1 ケーブル単体の節点座標、要素軸力、材料値の表。
- `温度荷重結果.xlsx`
  - 温度荷重ケース別の解析結果一式。
  - シート: `温度荷重なし`, `-30_-30`, `-30_-27`, `-30_-20`, `20_20`, `20_18`, `20_10`, `固定間距離比較`

## モデルの概要

- 座標系は `x, y, z` で、単位は mm。
- 表中の `y` は確認範囲ではすべて 0 で、2 次元の橋軸断面モデルとして整理されている。
- `温度荷重なし` シートの初期形状は 530 節点。
  - 全体の範囲: `x = -135100 .. 55100 mm`, `z = -1700 .. 50000 mm`
  - 主桁節点: 160 点、概ね `x = -135100 .. 55100 mm`, `z ≈ 0 mm`
  - 主塔節点: 15 点、概ね `x = 0 mm`, `z = -1700 .. 50000 mm`
- ケーブルは `C1` から `C12` までの 12 本として固定間距離が整理されている。
- `温度荷重なし` シートのケーブル軸力表は 360 要素分あり、単位は kN。

## 温度荷重結果の表構造

各温度ケースのシートは同じ列構成になっている。

| 列 | 内容 | 単位 |
| --- | --- | --- |
| A:D | 初期形状: 節点, x, y, z | mm |
| F:I | 変位: 節点, DX, DY, DZ | mm |
| K:N | 変形後形状: 節点, x, y, z | mm |
| P:R | ケーブル軸力: 要素, 断面力-I, 断面力-J | kN |
| U:X | 主桁節点座標: 節点, x, y, z | mm |
| Z:AC | 主塔節点座標: 節点, x, y, z | mm |
| AE:AK | 固定間距離計算用の節点座標と C1-C12 の固定間距離 | mm |

`固定間距離比較` シートには、温度ケースごとの C1-C12 固定間距離と、C1 の平均張力・相対誤差がまとめられている。
